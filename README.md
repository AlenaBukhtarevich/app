# Приложение для изучения иностранных слов по карточкам

<img src="..." alt="..." />

Главная страница, где отображаются списки слов всех существующих и карточек, а также режим тренировки.

У каждого слова есть основное значение, транскрипция, перевод, тема.
У каждого слова есть возможность его удаления и редактирования.
Есть возможность добавление слов.

<a href="https://vitejs.dev/">Документация</a> Vite.
